提案
=====
Openness Proposal は、任意の言語と複数のコンパイル済み言語をサポートするドキュメントであり、最終的には統一された形式でコミュニティにアップロードされます。
提案は、世界の国と地域のメンバーとネチズンによって提供されたアイデアを精緻化するためのチャネルであり、実行可能で参加可能な提案はすべて、評議会によって採択され、最終的に実現され、さまざまな数.さまざまな機能に従って評議会に与えられます.

提案编号
-----------
提案编号是在提案类型的前提下的延伸，联合现在提案编号通常由类型简写加之时间中的年（取最后两位数）、月组成作为年代索引并加上次号而定的，最后拼合为：``[type][year[day]][superposition]``。

子编号
-------
提案编号通常分为一级编号和二级编号，没有在编号中融入二级编号索引的默认为一级编号。二级编号通常作为一级编号的扩展，针对单独的问题所划分的编号，以帮助区分信息。

子编号是需要根据提案来进行申请，并最终由去中心化工作流中对文件编号系统(JNUN File numbering system) 收入的才可成为子编号进行使用。每个子编号占位一个格子。

即“``[alphabet]``” 一个编号中可以无限添加子编号以标识文件类型，最后拼合为：``[type][year[day]][superposition].[alphabet]···``。
